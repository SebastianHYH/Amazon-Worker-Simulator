# Amazon Worker Simulator

Proyek Unity yang mensimulasikan lingkungan gudang di mana pemain mengoperasikan lengan robot untuk mengelola barang di ban berjalan. Dibuat dari kolaborasi dengan Agate Academy dan ITB's Metaverse Lab. Mangga silakan lihat video demo di sini: [https://youtu.be/DCICUapagg4](https://youtu.be/DCICUapagg4).

## Fitur

* Kontrol lengan robot dengan mekanisme penjepit
* Dukungan VR untuk headset Oculus melalui XR Interaction Toolkit
* Sistem ban berjalan dengan pergerakan berbasis fisika
* Sistem pemunculan barang dan penilaian otomatis
* Lingkungan gudang interaktif

## Persyaratan Teknis

* Versi Unity: 6000.3.15f1
* Bahasa pemrograman: C#
* Sistem input: Unity Input System

## Struktur Proyek

* Assets/Models: Aset 3D untuk lengan robot dan gudang
* Assets/Scenes: Scene utama simulasi dan pengujian
* Assets/Scripts: Logika pergerakan, fisika, dan manajemen permainan
* Assets/Prefabs: Templat objek yang dapat digunakan kembali

## Instalasi

1. Klon repositori ini ke komputer lokal Anda.
2. Buka folder proyek menggunakan Unity Hub dengan versi 6000.3.15f1.
3. Buka scene yang terletak di Assets/Scenes/Main.unity.
4. Tekan tombol Play di Editor Unity.
