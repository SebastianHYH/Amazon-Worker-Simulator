# Amazon Worker Simulator

A Unity VR project simulating a warehouse environment where players operate a robotic arm to manage items on a conveyor belt. Made in collaboration with Agate Academy and ITB's Metaverse Lab. Have a look of the demo video here: [https://youtu.be/N2w5XAgxAs8](https://youtu.be/N2w5XAgxAs8).

## Features

* Robotic arm control with claw mechanics
* VR support for Oculus headsets via XR Interaction Toolkit
* Conveyor belt system with physics based movement
* Automated item spawning and scoring system
* Interactive warehouse environment

## Technical Requirements

* Unity version: 6000.3.15f1
* Scripting language: C#
* Input system: Unity Input System

## Project Structure

* Assets/Models: 3D assets for the robotic arm and warehouse
* Assets/Scenes: Primary simulation and testing scenes
* Assets/Scripts: Logic for movement, physics, and game management
* Assets/Prefabs: Reusable object templates

## Setup

1. Clone this repository to your local machine.
2. Open the project folder using Unity Hub with version 6000.3.15f1.
3. Open the scene located at Assets/Scenes/Main.unity.
4. Press the Play button in the Unity Editor.
